# Portfolio Project Summary for Design Review

## Project Overview
**Type:** Personal Portfolio Website
**Role:** Frontend-Focused Software Engineer
**Tone:** Modern, Premium, Professional, Tech-Forward (AI-Focused)

## Technology Stack
- **Framework:** Next.js 15 (App Router)
- **Styling:** Tailwind CSS v4 (using CSS variables and `@theme inline`)
- **Language:** TypeScript
- **Fonts:** Inter (Google Fonts)

## Current Design Aesthetic
- **Theme:** Deep Dark Mode (`bg-slate-950` / `#030712`)
- **Color Palette:**
  - **Primary:** Indigo 500 (`#6366f1`)
  - **Secondary:** Pink 500 (`#ec4899`)
  - **Accent:** Cyan 500 (`#06b6d4`)
- **Key Visual Elements:**
  - **Glassmorphism:** `bg-white/5 backdrop-blur-lg` used heavily on cards and headers.
  - **Gradients:** Text gradients (`bg-clip-text`) and background glow effects.
  - **Micro-interactions:** Hover effects on cards (translation/lift), smooth scrolling.
  - **Layout:** Responsive, single-page scroll structure.

## Structure & Component Breakdown

### 1. Header
- **Style:** Fixed, glassmorphic top bar.
- **Micro-interaction:** "Contact Me" button with hover state.
- **Navigation:** Smooth scroll links to About, Experience, Tech Stack, Projects.

### 2. Hero Section
- **Layout:** Split layout (Text Left, Image Right).
- **Key Elements:**
  - "Available for new opportunities" status badge.
  - Headline with gradient text on name "Haris V K".
  - Profile image with animated/glowing gradient ring.
  - Dual CTA buttons: "View My Work" and "View Resume".
- **Background:** Ambient gradient orbs (blurred blobs).

### 3. About Section
- **Content:** Professional summary focusing on Frontend, React, TypeScript.
- **Highlights:** Specific mentions of using AI tools like **Cursor** and **Google Antigravity**.

### 4. Tech Stack Section
- **Layout:** Grid of skill categories (Frontend, Backend, Tools).
- **Special Feature:** Dedicated "AI Technologies & Workflow" card.
  - Highlights **Cursor IDE** (Code generation, refactoring).
  - Highlights **Google Antigravity** (Autonomous agents).

### 5. Experience Section
- **Layout:** Vertical timeline with connecting lines.
- **Content:** Role at AptlyLabs.
- **Details:** Mentions Eco-friendly social platform, Redux, AWS, GraphQL.

### 6. Featured Projects
- **Layout:** 3-Column Grid.
- **Card Design:** Glassmorphic cards with image placeholders.
- **Unique Feature:** Distinct visual styling for "AI Projects" vs regular projects.
  - Badges like "Built with Cursor".
  - Hover effects revealing "View Live Project" links.

## Goal for Design Improvement
We want to take this foundation to the next level. We are looking for:
1.  **"Wow" Factor:** Advanced animations or transitions (e.g., Framer Motion).
2.  **Premium Feel:** Refined typography, spacing, or color usage.
3.  **Interactive Elements:** More engaging ways to showcase the "AI Workflow" distinction.
4.  **Layout Variations:** Breaking the standard grid/list monotony.
